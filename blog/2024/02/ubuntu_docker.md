## 概要

Ubuntu Serverインストール時にdockerをついでにインストールするとインストール自体はされているのだが使い勝手が悪い。

なのでその対応策。

### 対応策

以下のコードを上から順番に実行、最後だけパスワードが必要。

```zsh
sudo groupadd docker
sudo usermod -aG docker $USER
sudo chmod 666 /var/run/docker.sock # 再起動だけで済むかもしれない
```
